## 题库生成器使用说明

### 概述
题库生成器是一个将文本格式的单选题转换为JSON格式的工具，方便导入到对战游戏中使用。

### 功能特点
- 支持多种题目格式解析
- 自动识别题目、选项和答案
- 可视化预览和编辑
- 一键复制或导出JSON文件

### 使用方法

#### 1. 准备题目文本
将题目按照以下格式整理：

**格式一：标准格式**
```
1. 以下哪个不是JavaScript的数据类型？
A. String
B. Boolean
C. Float
D. Number
答案：C

2. CSS的全称是什么？
A. 层叠样式表
B. 计算机样式系统
C. 级联符号系统
D. 层次样式符号
正确答案：A
```

**格式二：紧凑格式**
```
1. HTML中用于定义超链接的标签是：
A. <link>
B. <href>
C. <a>
D. <url>
答案：C
```

#### 2. 解析题目
1. 打开 `题库生成器.html`
2. 在文本框中粘贴题目内容
3. 点击 **"解析题目"** 按钮
4. 查看解析结果和预览

#### 3. 校对答案
- 在题目预览区，每道题下方有答案选择下拉框
- 点击下拉框修改正确答案
- JSON输出会自动更新

#### 4. 导出题库
- **复制JSON**：点击"复制JSON"按钮，将内容复制到剪贴板
- **导出文件**：点击"导出JSON文件"按钮，下载 `.json` 文件

### JSON格式说明
```json
{
  "q": "题目内容",       // 题目的具体内容
  "o": [                // 选项数组，按顺序对应A、B、C、D选项
    "选项A内容",
    "选项B内容",
    "选项C内容",
    "选项D内容"
  ],
  "a": 0                // 答案索引，0=A, 1=B, 2=C, 3=D
}
```

## 单选题对战游戏使用说明

### 概述
单选题对战游戏是一款双人对战的答题竞技游戏，玩家通过抢答问题来攻击对方，先击败对手的一方获胜。

### 系统要求
- 现代浏览器（Chrome、Firefox、Edge等）
- 支持触屏操作（适配希沃白板等教学设备）

### 游戏流程

#### 1. 角色选择
1. 打开 `单选题对战游戏.html`
2. 在角色选择界面，P1和P2分别选择自己的角色
3. 点击 **"查看所有英雄"** 可浏览全部可选角色
4. 选择完成后点击 **"开始对战"**

#### 2. 战斗阶段
1. **题目展示**：屏幕中央显示题目和选项
2. **抢答环节**：
   - P1按键盘 **A键** 或点击左侧抢答按钮
   - P2按键盘 **L键** 或点击右侧抢答按钮
3. **答题环节**：抢答成功的玩家需要在规定时间内选择答案
4. **战斗反馈**：
   - 答对：攻击对方，造成扣血
   - 答错：自己扣血（自爆）
   - 超时：根据设置进行惩罚

#### 3. 游戏结束
- 任意一方血量降至0时游戏结束
- 显示胜利者和战斗统计
- 可选择 **"教师评讲"** 查看答题详情

### 教师面板功能

#### 进入教师面板
在角色选择界面点击 **"🎓 教师面板"** 按钮

#### 题库管理
1. **切换题库**：点击题库选择按钮切换不同题库
2. **新建题库**：点击"+ 新建题库"创建空白题库
3. **导入题库**：点击"+ 导入题库"上传JSON文件
4. **导出题库**：点击"📤 导出题库"保存当前题库
5. **删除题库**：点击"删除当前题库"移除不需要的题库

#### 题目编辑
1. **添加题目**：点击"+ 添加新题目"
2. **编辑题目**：点击题目列表中的编辑按钮
3. **删除题目**：在编辑界面点击删除按钮

#### 游戏设置
| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 每题答题时间 | 每道题的答题时限（秒） | 30秒 |
| 答对伤害值 | 答对时攻击对方的伤害 | 25点 |
| 答错自伤值 | 答错时自己受到的伤害 | 15点 |
| 抢答错误惩罚 | 抢答后答错的惩罚方式 | 无惩罚 |
| 抢答后答题时限 | 抢答后的答题时间（秒） | 10秒 |
| 超时未答惩罚 | 超时未答题的惩罚方式 | 扣血 |
| 超时扣血值 | 超时未答时扣除的血量 | 10点 |
| 音效开关 | 游戏音效的开启/关闭 | 开启 |

**注意**：修改设置后需要点击 **"保存设置"** 按钮生效

### 教师评讲功能

#### 进入评讲模式
游戏结束后点击 **"教师评讲"** 按钮

#### 功能说明
1. **答题记录列表**：左侧显示本局所有答题记录
2. **题目详情**：右侧显示选中题目的详细信息
   - 答题结果（正确/错误）
   - 题目内容
   - 四个选项（正确答案绿色高亮，错误选择红色标记）
   - 教师点评/知识点解析区域
3. **切换题目**：点击左侧列表中的题目进行切换

### 更新日志
在角色选择界面点击 **"📋 更新日志"** 按钮，可查看游戏的版本更新记录。

### 快捷键说明

| 操作 | P1 | P2 |
|------|-----|-----|
| 抢答 | A键 | L键 |

### 触屏操作说明
游戏完全支持触屏操作，适配希沃白板等教学设备：
- 点击按钮进行选择和确认
- 在"查看所有英雄"界面，点击上下滑动按钮浏览角色
- 所有按钮都支持触摸事件

### 注意事项
1. 建议使用最新版本的浏览器获得最佳体验
2. 游戏音效需要用户交互后才能自动播放
3. 题库数据会自动保存在浏览器本地存储中
4. 建议定期导出题库进行备份

### 常见问题

**Q: 游戏没有声音？**
A: 现代浏览器限制自动播放音频，需要点击任意位置激活音频。

**Q: 如何添加自己的题目？**
A: 使用题库生成器将文本题目转换为JSON格式，然后在教师面板导入。

**Q: 游戏数据会丢失吗？**
A: 游戏数据保存在浏览器本地，清除浏览器数据会导致丢失，建议定期导出备份。

**Q: 支持多少道题目？**
A: 理论上没有限制，但建议每个题库保持在100道题以内以保证加载速度。
