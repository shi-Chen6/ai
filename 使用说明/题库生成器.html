单选题转JSON格式转换器

题目预览：
JSON输出：
JSON字段说明：
{
  "q": "题目内容",       // 题目的具体内容
  "o": [                // 选项数组，按顺序对应A、B、C、D选项
    "选项A内容",         // A选项内容
    "选项B内容",         // B选项内容
    "选项C内容",         // C选项内容
    "选项D内容"          // D选项内容
  ],
  "a": 0,               // 答案索引，0=A, 1=B, 2=C, 3=D
  "analysis": ""        // 解析内容（可选），对题目的详细解释说明
}

示例：

{
  "q": "以下哪个不是JavaScript的数据类型？",
  "o": [
    "String",
    "Boolean",
    "Float",
    "Number"
  ],
  "a": 2,  // 表示正确答案是选项C
  "analysis": "JavaScript的数据类型包括：String（字符串）、Boolean（布尔值）、Number（数值，包括整数和浮点数）等。Float不是JavaScript的独立数据类型，浮点数统一用Number表示。"
}
