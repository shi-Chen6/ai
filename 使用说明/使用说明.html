题库生成器使用说明

概述

题库生成器是一个将文本格式的单选题转换为JSON格式的工具，方便导入到对战游戏中使用。

功能特点
  • 支持多种题目格式解析
  • AI智能排版：接入 DeepSeek 大模型，自动将不规范题目文本标准化排版
  • 自动识别题目、选项和答案
  • 智能提取题目解析和答案标注
  • 可视化预览和编辑
  • 一键复制或导出JSON文件
使用方法

1. 准备题目文本

格式一：标准格式

1. 以下哪个不是JavaScript的数据类型？
A. String
B. Boolean
C. Float
D. Number
答案：C

2. CSS的全称是什么？
A. 层叠样式表
B. 计算机样式系统
C. 级联符号系统
D. 层次样式符号
正确答案：A

格式二：紧凑格式

1. HTML中用于定义超链接的标签是：
A. <link>
B. <href>
C. <a>
D. <url>
答案：C

2. 一键排版（AI智能格式化）

如果题目格式不统一、不规整，可直接使用 一键排版 功能：

  1. 将任意格式的题目文本粘贴到输入框
  2. 点击 一键排版 按钮
  3. 系统调用 DeepSeek AI大模型 自动识别题目结构，将不规范格式统一标准化
  4. AI会自动：提取答案标注、拆分同行的选项、保留解析说明、统一序号和选项格式
  5. 等待数秒即可获得排版整齐的规范题目文本

3. 解析题目

  1. 排版完成后，检查预览效果
  2. 点击 解析题目 按钮
  3. 查看解析结果和预览

4. 校对答案

在题目预览区，每道题下方有答案选择下拉框，点击下拉框修改正确答案，JSON输出会自动更新。

5. 导出题库

  • 复制JSON：点击按钮将内容复制到剪贴板
  • 导出JSON文件：下载 .json 文件
JSON格式说明
{
  "q": "题目内容",
  "o": [
    "选项A内容",
    "选项B内容",
    "选项C内容",
    "选项D内容"
  ],
  "a": 0
}

a 索引：0=A, 1=B, 2=C, 3=D

单选题对战游戏使用说明

系统要求

现代浏览器（Chrome、Firefox、Edge等），支持触屏操作（适配希沃白板等教学设备）。

角色选择
  1. 打开 单选题对战游戏.html
  2. P1和P2分别选择自己的角色
  3. 点击 查看所有英雄 浏览全部可选角色
  4. 选择完成后点击 开始对战

新增功能

  • 音量调节：点击 🔊 音量调节 按钮，可独立调节选角BGM、战斗BGM、系统音效和英雄语音音量
  • 使用说明：点击 📘 使用说明 按钮，打开详细的使用指南文档
  • 音效配置：点击 🎵 音效配置 按钮，为英雄自定义进攻和答错音效
  • 更新日志：点击 📋 更新日志 按钮，查看版本迭代信息
战斗阶段
  • 题目展示：中央显示题目和选项，抢答前选项AB在一行CD在一行，居中显示
  • 抢答环节：P1按键盘 A 或点击左侧按钮；P2按 L 或右侧按钮
  • 答题环节：抢答成功的玩家在规定时间内选择答案，答题框采用两栏布局，提供更充足的显示空间
  • 战斗反馈：答对攻击对方；答错自己扣血；超时根据设置惩罚

答题界面优化

  • 答题框宽度扩展至1100px，提供更充足的显示空间
  • 选项采用两栏布局，提升空间利用率
  • 选项字体大小优化为24px，配合两栏布局提升阅读体验
  • 四个选项使用四种颜色区分，方便学生直观观察
  • 选项A与选项C左对齐，选项B与选项D左对齐

游戏结束

任意一方血量归零，显示胜利者和统计，可点击 教师评讲 查看答题详情。

教师评讲功能

游戏结束后点击 教师评讲 进入评讲模式。

  • 左侧答题记录列表
  • 右侧显示题目详情：结果、题目、选项（正确答案绿色高亮，错误选择红色标记）、教师点评区域
快捷键说明
操作P1P2
抢答AL
触屏操作说明

完全支持触屏，适配希沃白板：点击按钮进行选择和确认；在“查看所有英雄”界面滑动浏览；所有按钮支持触摸事件。

注意事项
  • 建议使用最新版浏览器
  • 游戏音效需要用户交互后才能自动播放（点击任意位置激活）
  • 题库数据保存在浏览器本地存储中，定期导出备份
常见问题 (点击展开)

Q: 游戏没有声音？
A: 现代浏览器限制自动播放，点击任意位置激活音频。可通过音量调节功能调整各类型音量。

Q: 如何添加自己的题目？
A: 使用题库生成器转换为JSON，然后在教师面板导入。

Q: 游戏数据会丢失吗？
A: 保存在浏览器本地，清除数据会导致丢失，建议定期导出备份。

Q: 支持多少道题目？
A: 理论上不限，建议每个题库保持在100道以内以保证加载速度。

Q: 如何自定义英雄音效？
A: 点击音效配置按钮，为英雄上传本地音频文件作为进攻和答错音效。

Q: 答题框显示异常怎么办？
A: 系统已优化答题框显示，支持长题目内容完整显示，如仍有问题请刷新页面。

Q: AI智能分析和AI排版如何使用？
A: 两者均使用DeepSeek大模型驱动。数据大屏中点击右下角"AI智能分析"按钮即可获得数据洞察报告；题库生成器中点击"一键排版"即可自动标准化题目格式。需要网络连接。

AI 智能分析功能

概述

本项目在两个关键环节接入了 DeepSeek 大语言模型，分别用于数据智能洞察和题目智能排版，大幅提升了教师的工作效率和数据分析深度。

数据大屏 · AI智能洞察

功能入口

打开 数据大屏.html，点击右下角紫色悬浮按钮 ✨ AI 智能分析

AI分析内容

系统自动采集当前数据大屏中的所有核心指标（参与玩家数、整体正确率、总对战场次、TOP3胜场玩家、TOP3易错题目等），发送给 DeepSeek 大模型进行深度分析，输出包含三个模块的结构化报告：

  • 业务洞察：基于数据提炼关键趋势和发现
  • 问题诊断：识别当前教学中的潜在问题区域
  • 优化建议：给出针对性的教学策略改进建议

教师可以在弹窗中阅读 AI 生成的分析报告，也可以点击 重新分析 刷新结果。分析结果以 Markdown 格式排版，清晰易读。

题库生成器 · AI智能排版

功能入口

打开 题库生成器.html，在文本框粘贴任意格式的题目后，点击 一键排版

AI排版能力

教师输入的题目格式往往千差万别（选项混在同一行、答案标注不明、序号缺失等）。AI排版调用 DeepSeek 大模型，自动完成以下标准化处理：

  • 统一题目序号格式（数字 + 点号）
  • 将同行选项拆分为4行独立选项
  • 自动提取答案标注（如"答案：C"）
  • 保留原题中的填空括号
  • 提取并保留题目解析/分析说明
  • 题目间自动添加空行分隔
使用技巧：即使格式非常杂乱的Word文档题目，AI排版也能识别并标准化。排版完成后可直接点击"解析题目"进行JSON转换。
